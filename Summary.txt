A Django-based web application that manages services, customers, and bookings. Key features include:

Service Management

Create, update, and delete services.

Each service has a title, description, price, and availability date.

Customer Management

Add and manage customers.

Track customer details and booking history.

Service Booking

Customers can book available services.

Single-date availability per service to prevent double-booking.

User Authentication

Superusers can manage services and customers.

Login required for administrative actions.

Admin Interface

Manage services, customers, and bookings via Django admin.

HTMX/Frontend Features

Interactive UI for booking and managing services.

Responsive layout with images above service descriptions.
